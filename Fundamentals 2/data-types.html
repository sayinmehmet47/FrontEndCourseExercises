JavaScript Data Types And Data Structures

Data Types

# Type Description Example
1 String Stores textual expressions in string data type. When entering string data into variables, (") quotation marks or (‘) apostrophes are used. let name=”life code”;
3 Number Store number expressions.There is no need for ("") double quotes or (") single quotes when defining values let a=10; var pi=3.14
3 Boolean Some expressions have only two alternatives. The expression is either true (true) or false (false). Such variables are called boolean expressions. Logical data are used for expressions that require conditions. var a = true;var b = false;
4 Objects Objects keep their sub-variable data. Use curly brackets when ordering child variables. Variables are written separated by commas. let info = { name:”lifeacode”, content:”education” };
5 Undefined There are variables that do not contain values in the JavaScript language. These variables do not contain data. When creating the variable that does not contain data, it is created as a var variable. So only the variable name is written. The (=) equals operator is not used. let city;
6 Null In some cases, the data in the variable must be deleted. Null expression is used for this. city=null;
7 typeof The type of a variable can be queried using the typeof operator. typeof “lifeacode” //string typeof “123” //number typeof true //boolean typeof [1855,4785,5412,5698] //object
